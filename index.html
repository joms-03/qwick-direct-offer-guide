Sending a Direct Offer to Fill a Shift

A step-by-step walkthrough for finding qualified pros in Command Center and pushing a Direct Offer to fill an unfilled or under-staffed shift — plus a Tableau-based alternative for when you need a bigger custom list.

⏱ ~12 minutes 🔑 Admin access required 🧰 Slack + Qwick Admin + Command Center
1. Claim the Slack request 2. Find the shift 3. Open Command Center 4. Read the diagnostics 5. Filter candidates 6. Send offers in bulk Alt: Open Pro Export Alt: Filter market & skill Alt: Set quality filters Alt: Export the list Alt: Grab the Pro IDs Alt: Open Shift Options Alt: Paste & send 14. Confirm it worked

Before you start

What you'll need

  • Access to the #magic-pro-support Slack channel
  • Access to the shift in View Shifts (biz.qwick.com/admin)
  • Admin permissions to open Command Center and send a Direct Offer
  • Access to the Pro Export Tableau dashboard — only needed for the alternative path below

When to use this

  • A "Fill a Shift Request" lands in #magic-pro-support, or a shift is showing as Unfilled
  • Start with Command Center — it lives inside the admin site, needs no separate license, and already surfaces a scored, filterable pool of candidates
  • Fall back to the Tableau path only if Command Center's pool doesn't have enough qualified pros, or you need a much broader custom list
1

Claim the request in Slack

Fill requests often come in through #magic-pro-support as a "V3 Fill a Shift Request" workflow message. It shows who submitted it, a link straight to that shift's Applicants page, the Auto-Select / Overfill settings, and any special notes (e.g. "boost and max offers").

3

Open Command Center

From the shift row, click the ⚙️ gear icon to open Shift Options, then under the Admin-only section click "Open in Command Center." This drops you into a live, per-shift dashboard — no Tableau license required, since it's built directly into the admin site.

5

Filter the candidate pool

Scroll down to Possible Matches and click Filters. Set a minimum match score, a maximum distance, a minimum average rating, a minimum number of paid gigs, and a minimum cross-shift coefficient — this is the same quality bar Tableau's sliders gave you, built right into the candidate list.

6

Select pros and send offers in bulk

Sort by Combined score, then either click Send DO on individual rows, or check the boxes next to several candidates and use the Send DO button at the top to blast offers to everyone selected at once — no CSV, no copy-pasting IDs.

7

Open the Pro Export dashboard in Tableau

In a new tab, go to Explore → Qwick Internal → Pro Health → Pro Tracking and select the Pro Export tab. This dashboard lets you build a custom list of pros filtered by market, skillset, and quality metrics.

13

Paste the Pro IDs and send the offer

In the Admin offer modal, paste the Pro IDs you copied in Step 6 into the Pro IDs box — duplicates are removed automatically. Add an optional Bonus to sweeten the offer, review the checkboxes, then click Send offer.
